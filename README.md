#spring-microservices-kafka

Projeto para estudo da arquitetura em microserviços com duas aplicações integradas com Apache Kafka.

O projeto do Bootcamp foi a criação de um Ecommerce dividido em 3 partes.

    1 - microservico ecommerce-checkout-api 
    2 - microservico ecommerce-payment-api
    3 - ecommerce-checkout-frontend

O projeto tem o seguinte setup:

    -Java 15
    -Spring Boot 2.4.4
    -Spring Cloud Streams
    -Docker 
    -Zookeeper
    -kafka
